# 🚗 ConsumoMaster

Um aplicativo web moderno para simular e analisar o consumo de combustível.

### 🛠️ Tecnologias
* **Python / Flask** (Backend)
* **Selenium** (Web Scraping de preços reais)
* **Bootstrap 5** (Interface responsiva)

### 🚀 Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o app: `python app.py`
